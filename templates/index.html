Local Development Setup

The backend is configured to use local directories:

  • uploads/ - For uploaded videos
  • captions/ - For caption data
  • outputs/ - For processed videos

Frontend Connection

The frontend should be configured to connect to this local backend at http://localhost:8080.

Check the following files in the frontend:

  • Clip/App/.env - Should have VITE_API_BASE_URL=http://localhost:8080
  • Clip/App/src/lib/config.js - Should use the environment variable or fallback to localhost
  • Clip/App/vite.config.js - Should proxy API requests to localhost:8080
